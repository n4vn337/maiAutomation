# maiAutomation
Scans included
  1. nmap with -sC -sV and -oA flags
  2. nikto /- simple host scan
  3. gobuster for directory busting
 
 ### prerequisites				
  gnome-terminal
  `sudo apt-get install gnome-terminal`
  
	
### Now we are good to go
	
Usage
	`git clone https://github.com/n4vn337/maiAutomation`
	`cd maiAutomation`
	`chmod +x script.sh`
        `./script.sh ip-addr`
